## How to run

```bash
$ git clone https://github.com/pabloVinicius/covid-19-dashboard.git

$ cd covid-19-dashboard

$ yarn

$ yarn dev
```

> Open [http://localhost:3000](http://localhost:3000) to see the app running in dev mode.<br>
